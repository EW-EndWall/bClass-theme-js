# Before after function

Item css is add before

## Before add class

### example

```html
<div
  data-ba="before"
  data-ba-content="Before Content"
  data-ba-class="m-auto"
></div>
```

to

```css
cutom-create8546511:before {
  margin: auto;
}
```

## After add class

### example

```html
<div
  data-ba="after"
  data-ba-content="After Content"
  data-ba-class="m-auto"
></div>
```

to

```css
cutom-create15661656:before {
  margin: auto;
}
```
